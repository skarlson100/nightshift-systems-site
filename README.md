# Nightshift Systems

A static landing page for Nightshift Systems, built with Next.js App Router, TypeScript, and Tailwind CSS.

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Deployment

```bash
git init
git add .
git commit -m "Initial Nightshift Systems landing page"
```

Then:

1. Push the repository to GitHub.
2. Import the GitHub repository into Vercel.
3. Add `nightshiftsystems.com.au` as the custom domain in Vercel.
4. Update your domain DNS records using the values provided by Vercel.
